# 📊 Student Performance Prediction (ML Project)

## Overview

This project analyzes student performance and builds a machine learning model to predict Math scores based on Reading and Writing scores.

## Steps Performed

* Data loading and cleaning
* Exploratory Data Analysis (EDA)
* Visualization
* Feature selection
* Model training using Linear Regression
* Model evaluation

## Results

* Model evaluated using Mean Squared Error and R² Score
* Demonstrates relationship between academic subjects

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
